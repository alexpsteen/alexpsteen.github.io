Forecast



Descripton:

This SPA web application allows for the user to choose from a selection of cities and view the 7 day weather forecast. THe webpage uses Web APIs and XMLHTTP requests through JavaScript to obtain the data.


Date:

March 2017 (1 month)


Language:

Javascript, XML, HTML5, CSS


Github link

This is the main screen for the Weather Forecast application, showing the highs and lows and graphics for the types of expected weather. There are also a display of the days of the week up to the next week.

Here is the drop down list of the major cities that are available to look at the forecast for. These can be easily changed by changing the city abbreviation to the desired city in the XMLHTTP request.
