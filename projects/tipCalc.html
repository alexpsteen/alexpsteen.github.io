Tip Calc



Descripton:

This Android App allows the user to calculate a tip.


Date:

October 2016 - November 2016 (1 month)


Language:

Java, XML, Android Studio


Above is the Main page of the TipCalculator Android Application.

The image above shows an example screen of the tip calculations. The user inputs a subtotal and a tip amount and also has the option to change the percentage of tip given as well as the tax of the restaurant/service. Both the Tip total and the Grand Total are computed so that the user knows how much to write down for each section.

This is a screen showing an example of entering a custom tip. Once the button is selected, the tip amount is calculated according to it and all of the information is updated automatically after input.
